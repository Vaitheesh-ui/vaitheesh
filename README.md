̣ Alex Carter Portfolio

A modern frontend developer portfolio built with React, Tailwind CSS, Vite, React Router, and Framer Motion.

## Features

- Responsive 5-page portfolio: Home, About, Skills, Projects, Contact
- Animated page transitions and scroll reveal effects
- Glassmorphism cards, gradient glows, and soft shadows
- Smooth sticky navbar with mobile menu
- Animated typewriter hero text and floating icon accents
- Interactive project cards and contact form

## Setup

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
